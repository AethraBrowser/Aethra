# Aethra Browser

**The browser that gets out of the way.**

Aethra is a modern web browser focused on speed, privacy, and intelligent workflows — built for people who are tired of fighting their tabs and their tools.

---

## ✨ Features

### ⚡ Speed & Performance

* Near-instant page loads with smart prefetching
* Minimal RAM usage — tabs don’t eat your system alive
* Fast cold-start times

### 🔒 Privacy by Default

* Built-in tracker and ad blocking
* DNS-over-HTTPS enabled out of the box
* Fingerprint resistance
* Ephemeral mode that actually leaves **no trace**

### 🧠 Smart Browsing

* Unified search across tabs, history, and bookmarks
* Natural language queries
  *“that article about birds I read last week”*
* Inline tools: calculations, conversions, time zones

### 🧩 Tab Management (finally fixed)

* Vertical tab sidebar
* Tab grouping + collapsing
* Instant tab search
* Auto-suspend inactive tabs
* Saved sessions & workspaces

### 🛠 Built-in Tools

* Screenshot + annotation
* Distraction-free reading mode
* Clean PDF viewer
* Media downloader
* Built-in VPN / proxy support
* Password manager with passkeys

### 👨‍💻 Developer Experience

* Powerful DevTools
* Built-in HTTP client / API tester
* Local file overrides
* Accessibility audits

### 🎨 Customization

* Full UI theming (not just colors)
* Per-site settings
* Fully remappable keyboard shortcuts

### 🔄 Sync & Cross-Device

* End-to-end encrypted sync
* Seamless device handoff
* Open sync protocol (no lock-in)

## 🎯 Philosophy

Aethra is built on a simple idea:

> The best browser is the one you don’t notice.

No clutter. No dark patterns. No unnecessary friction.
Just fast, private, effortless browsing.
